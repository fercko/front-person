## Iniciar el proyecto

Requiere:
```
    node v20
```
primero, correr uno de los siguientes comandos:

```bash
(solo la primera vez)
npm install
#or
yarn install

Luego:
npm run dev
# or
yarn dev
# or
```

Abrir [http://localhost:3000](http://localhost:3000) en cualquier navegador.

Para modificar la url del backend, abrir el archivo "next.config.js" y modificar "url_back".
